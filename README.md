# Relatório Gerencial
Este relatório foi desenvolvido com o objetivo de oferecer uma visão clara, dinâmica e interativa sobre o desempenho da empresa.

Na primeira visão, o usuário pode explorar os dados a partir de diferentes categorias, como segmento, país ou categoria de produto. Essa abordagem permite identificar padrões específicos de cada área do negócio, facilitando análises direcionadas e decisões estratégicas.

Já na segunda visão, o foco está nas métricas financeiras e operacionais. O usuário pode alternar entre indicadores como vendas, lucro ou quantidade vendida, permitindo uma análise comparativa entre diferentes KPIs. Essa flexibilidade ajuda a entender não apenas o volume de vendas, mas também a rentabilidade e eficiência operacional.

Ambas as visões seguem a mesma identidade visual, garantindo consistência e uma experiência fluida. O resultado é um relatório intuitivo, poderoso e alinhado às melhores práticas de análise de dados
